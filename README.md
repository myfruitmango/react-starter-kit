# Your Project Awesome🚀

## ✨ Starting
using npm:
```bash
npm install
```
or if you using yarn:
```bash
yarn install
```

## 🐢 Run
For development in local
```bash
pm2 start pm2.dev.config.js
```
or
```bash
npm start
```

## 🍔 Build
Deploy
```bash
npm run build
```
